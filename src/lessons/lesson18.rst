Lesson 18: Indexing
===================

Learning Objectives
-------------------

* Learn how indexes can make queries run faster and why
* Understand the types of indexes

  * Clustered
  * Non-clustered
  * Column store
  * Others for special data types

* Create an index
* Drop an index
